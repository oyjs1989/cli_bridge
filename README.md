# cli-bridge

Multi-channel AI Assistant gateway powered by Claude CLI.

Bridges chat platforms (Telegram, Discord, Slack, Feishu, DingTalk, QQ, WhatsApp, Email, Mochat) to an AI backend (Claude CLI or iflow CLI).

## Quick Start

```bash
uv sync
uv run cli-bridge gateway run
```

Config file: `~/.cli-bridge/config.json`
